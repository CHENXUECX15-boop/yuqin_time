# 企效工作台

这是一个企业员工版工作台静态网页原型，入口文件为 `index.html`。

## 当前包含

- 今日总览：打卡、专注、任务、会议、今日时间块
- 工作节奏：到位/离开/请假、专注计时、近 7 天趋势
- 任务看板：待办、进行中、已完成
- OKR 与项目：个人目标进度、数字输入/滑块双向调整、项目组合
- 会议协作：会议结论与行动项
- 向上管理：进展、风险、需要支持
- 健康边界：能量、压力、准点下线
- 每日复盘：成果、风险、明日重点
- 数据看板：工作趋势、任务分布、员工体感
- 数据管理：本地 JSON 导入导出、小程序迁移集合草案

## 数据说明

当前版本使用浏览器 `localStorage` 保存数据，不依赖后端。后续迁移微信小程序时，可将 `app.js` 中的状态对象拆成小程序 `pages`、`components` 和云开发集合。
