# 早日成为富婆

work for tomorrow.

这是一个个人主理人工作台静态网页原型，入口文件为 `index.html`。

## 当前包含

- 多主题外观：默认简约黑色，可切换彩色猫猫、线条猫猫、多巴胺、红色、黄色和绿色单色
- 中英文界面切换：可在数据管理中切换语言，偏好保存在本地
- 今日总览：开工、专注、实时推进卡、任务、会议、今日时间块
- 工作节奏：开工/收工/休整、按日期分组的行动记录、专注目标与奖励、专注历史、近 7 天趋势
- 任务看板：待办、进行中、已完成
- 时间表：以 0.5 小时为最小刻度，手动编辑一天 24 小时的活动安排，并支持日历导入
- 会议协作：会议结论、行动项和可勾选的常用联系人
- 每日复盘：成果、风险、明日重点、五档情绪满意度和极端情绪原因
- 数据看板：工作趋势、任务分布
- 数据管理：本地 JSON 导入导出

## 数据说明

当前版本使用浏览器 `localStorage` 保存数据，不依赖后端。
